## 📎 프로젝트 소개
<img width="1493" alt="스크린샷 2023-01-16 오후 8 14 42" src="https://user-images.githubusercontent.com/93763666/212665462-0f516964-8442-4098-8e8b-4e6399fde5be.png">

수산물을 시장에서 직접 사먹는 사람들을 위한 서비스
<br/><br/>


##### 사먹사는 예산과 인원만으로 적당한 횟감을 제안합니다.

수산물 가격을 조사하고 어종마다 다른 수율을 일일이 확인해 볼 필요없이 사용자가 인원과 예산만 입력해주시면 현 시세에 맞게 구매 가능한 횟감을 골라줍니다.
